
# RoutePulse — Operations Cockpit

An operations dashboard for regional logistics managers, designed to
answer four questions in under 30 seconds each morning:

- What requires my approval?
- Which anomalies need attention?
- What is current operational health?
- What should I act on first?

Built as a calm, minimal cockpit — not a traditional analytics dashboard.

## Tech Stack

- React + Vite
- Tailwind CSS
- JavaScript (no TypeScript)
- No backend — mock JSON data sources

## Getting Started

```bash
npm install
npm run dev
```

Runs at `http://localhost:5174` by default.
